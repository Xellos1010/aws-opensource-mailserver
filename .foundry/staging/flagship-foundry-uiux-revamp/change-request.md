# Change Request

## Request
Create a specialized AI agent pipeline that can diagnose and design the UI/UX requirements for:
- a new system setup,
- a new feature request, and
- an existing system that needs a UI/UX upgrade.

The immediate target is the Flagship Foundry / Cursor Companion ecosystem, which already has strong SDLC primitives but currently feels too menu-driven, too basic visually, and too weak at fast navigation, graph interaction, panel control, and drill-down from topology to pseudocode, diffs, work orders, sessions, and evidence.

## User-visible problems to solve
- Operators have to dig through too many menus to find critical actions.
- Graphs are visible but not fully usable as an interactive navigation substrate.
- Quick actions are not surfaced according to intent and urgency.
- Panels cannot yet be treated as first-class operator tools.
- The user cannot move smoothly from ecosystem overview to module internals and execution evidence.
- The system needs a stronger chat-driven orchestration layer for creating or suggesting agents, pipelines, features, and project coordination work.

## Desired outcome
Deliver a reusable UI/UX planning and architecture system plus a specific Foundry UI/UX revamp packet that can be imported into the next workspace and executed as bounded work orders.
